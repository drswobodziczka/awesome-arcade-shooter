# Game Assets Guide

## Free Asset Sources

### Top Sites

- **[Kenney.nl](https://kenney.nl)** - Massive free asset packs (pixel art, vectors, sounds)
- **[OpenGameArt.org](https://opengameart.org)** - CC0/CC-BY community assets
- **[itch.io/game-assets](https://itch.io/game-assets)** - Free + PWYW assets
- **[CraftPix.net](https://craftpix.net/freebies)** - Free 2D sprite section
- **[GameArt2D.com](https://gameart2d.com/freebies.html)** - Free sprite packs

### Search Terms

For our arcade shooter:
- "shmup sprites"
- "top-down shooter"
- "plane sprites" / "airplane sprites"
- "apocalyptic tileset"
- "scrolling background"

## AI Asset Generators

### Recommended Tools (2025)

1. **[Scenario.gg](https://scenario.com)** - Train custom models, consistent style
2. **[Layer.ai](https://layer.ai)** - 2D/3D, pipeline integration
3. **[Leonardo.ai](https://leonardo.ai)** - Best quality/value balance
4. **[PixelVibe (Rosebud AI)](https://lab.rosebud.ai/ai-game-assets)** - Quick 2D sprites
5. **Stable Diffusion** (ComfyUI/Automatic1111) - Full control, unlimited, requires setup

### Quick Start

**For fast prototyping:**
- Leonardo.ai or Scenario - best for sprite generation without tech setup
- Kenney.nl - ready-made packs for immediate use

**For production:**
- Scenario.gg - train model on your art style for consistency
- Layer.ai - integrates into game pipelines

## Asset Needs

### Player
- Airplane sprite (top-down view)
- Animation frames (optional: banking left/right)

### Enemies/NPCs
- Enemy aircraft variations (2-3 types minimum)
- Animations (optional: damage states)

### Environment
- Apocalyptic scrolling background
- Tileable vertical scroll
- Parallax layers (optional)

### Effects (Future)
- Explosions
- Projectiles/bullets
- Power-ups
